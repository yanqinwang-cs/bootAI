# Historical OpenRouter Code Assistant

This directory preserves an earlier code-assistant exercise that used OpenRouter through the OpenAI Python client. It is historical, unsupported, and unrelated to bootAI's local file-organizer architecture.

The code is intentionally retained without functional changes. It is excluded from bootAI package discovery and runtime dependencies. In particular, bootAI does not install or configure the `openai` or `python-dotenv` packages and does not require an OpenRouter API key.

Anyone intentionally inspecting this historical exercise must use a separate environment and manage its obsolete dependencies and credentials independently. Do not treat this directory as a bootAI entry point, supported example, or cloud integration.
