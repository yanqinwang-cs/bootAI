# Prompt Patterns

## Useful Patterns

- Schema-first prompting.
- Path-preservation prompting.
- Allowed-value prompting.
- Low-temperature local generation.
- Advisory-only prompting.
- Compact metadata payloads.

## Bad Patterns

- Broad open-ended prompts.
- Asking for file actions.
- Asking for subjective cleanup judgments.
- Free-form prose when structured output is required.
- Asking the model to infer filesystem facts that Python already knows.
